## RECORDING TIPS
- **Frame Rate**: 60fps for action shots (can slow down in post)
- **Resolution**: 4K if possible (gives room to punch in)
- **Lighting**: Soft, diffused light for hardware shots
- **Stability**: Use tripod for screen recordings
- **Screen Recording Settings**: 
  - Hide desktop clutter
  - Increase terminal font size (16-18pt minimum)
  - Use high contrast themes
  - Slow down cursor movements
  - Hide notifications/distractions
